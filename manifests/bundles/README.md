# Bundle Manifest Class

`pkt-r06-04` establishes `bundles/` as an active target class.

`bundle-closure.manifest.json` materializes the first bundle control object for the operational ChatGPT packet pipeline and captures the selective closure surface used to rebuild `chatgpt-pipeline.bundle.tgz`.
`kernel-workflow-closeout.manifest.json` is a closeout checklist control object for the kernel-spec-defined source-to-render path. It is not a bundle artifact; it validates that the operational workflow mirrors the spec-defined authority model, pipeline, gates, and Jsonnet rendering lane.
